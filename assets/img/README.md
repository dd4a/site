# DD4A Image Assets

This folder contains brand and UI image assets for **Digital Design 4 All (DD4A)**.

## Brand Names
- Use **Digital Design 4 All** (or **DD4A**) in UI and general copy.
- Use **Digital Design 4 All Incorporated** only where legally required (e.g., privacy/legal copy).

## Colours
- Primary navy: `#102070`
- Secondary slate: `#283850`
- White: `#FFFFFF`

## Logos (SVG)
- `d4a_mark_blue.svg`: Main logo mark (navy fill)
- `d4a_mark_white.svg`: Main logo mark (white fill, for dark backgrounds)
- `d4a_mark_currentColor.svg`: Main logo mark (inherits CSS color)

## Emblems (SVG)
- `d4a_badge_darkmode.svg`: Emblem for dark mode (auto color via CSS)
- `d4a_badge_light.svg`: Emblem for light backgrounds

## Icons
- `favicon.ico`: Browser favicon (multi-size)
- `favicon.svg`: SVG favicon for modern browsers
- `icon-192.png`, `icon-512.png`: PNG icons for PWA/manifest
- `apple-touch-icon.png`: Icon for iOS home screen

## Web Manifest
- `site.webmanifest`: PWA manifest file

## Notes
- All SVGs are accessible and include ARIA labels and titles.
- PNGs and SVGs are referenced in site headers, favicons, and manifest files.
