# DD4A image assets

This folder contains brand and UI image assets for **Digital Design 4 All (DD4A)**.

## Brand names
- Use **Digital Design 4 All** (or **DD4A**) in UI and general copy.
- Use **Digital Design 4 All Incorporated** only where legally required (e.g., privacy/legal copy).

## Colours
- Primary navy: **#102070**
- Secondary slate: **#283850**
- White: **#FFFFFF**

## Logos
### Horizontal wordmark
- `logo-horizontal-white.png`  
  Use on dark backgrounds (e.g., site header).

- `logo-horizontal.png`  
  Use on light backgrounds or white panels.

### Emblem
- `emblem-navy.png`  
  Use on light backgrounds (icons, badges, small marks).

- `emblem-white.png`  
  Use on dark backgrounds.

## Icons
- `favicon.ico`  
  Browser favicon (multi-size).

- `favicon-32.png` (and other sizes if present)  
  PNG favicon for modern browsers.

- `apple-touch-icon.png`  
  Icon for iOS home screen.

## Social preview
- `og-card.svg`  
  Open Graph preview image used for link sharing. If you replace this, keep the same filename or update the `<meta property="og:image">` tag site-wide.
